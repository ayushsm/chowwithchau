# Chow with Chau - Brand Colors

This document defines the official color palette for the Chow with Chau website.

## Color Palette

### Sage Green Tones

**Dark Sage Green**
- Hex: `#7E8C69`
- RGB: `rgb(126, 140, 105)`
- Use: Primary accents, headings, buttons

**Light Sage Green**
- Hex: `#9CAD8C`
- RGB: `rgb(156, 173, 140)`
- Use: Secondary accents, hover states

### Pink Tones

**Pink**
- Hex: `#F6B0BB`
- RGB: `rgb(246, 176, 187)`
- Use: Call-to-action elements, highlights

**Light Pink**
- Hex: `#F1C8CB`
- RGB: `rgb(241, 200, 203)`
- Use: Soft backgrounds, borders

### Neutral

**Cream/Beige**
- Hex: `#ECE0DA`
- RGB: `rgb(236, 224, 218)`
- Use: Backgrounds, cards, subtle elements

**Use Cases:**
- Gradient backgrounds or transitions
- Chart visualizations requiring multiple shades
- Hover state variations
- Border and divider colors with subtle variation

## Usage Guidelines

- Use sage green tones for primary branding elements
- Use pink tones for interactive elements and calls-to-action
- Use cream/beige for backgrounds to create warmth and softness
- Maintain sufficient contrast for accessibility

## Color Combinations

### Recommended Pairings
- Dark Sage Green (#7E8C69) with Cream (#ECE0DA)
- Pink (#F6B0BB) with Dark Sage Green (#7E8C69)
- Light Pink (#F1C8CB) with Light Sage Green (#9CAD8C)
